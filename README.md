# Simple To-Do List App

This is a clean and interactive To-Do List application built using **HTML**, **CSS**, and **JavaScript**.  
It helps users manage daily tasks efficiently by allowing them to add, mark, and remove tasks.

---

## Live Demo
View the app here:  [Live Demo](https://kirtinanwanii.github.io/Productivity-To-Do-App/)  

---

##  Features

-  Add new tasks  
- Mark tasks as completed  
- Delete tasks  
-  Simple and user-friendly interface  
- Responsive design  

---

##  Technologies Used

- **HTML** – Structure of the app  
- **CSS** – Styling and layout  
- **JavaScript** – Task logic and DOM manipulation
